# Marmalade

A lazy orange tabby companion with heavy-lidded charm, snack-first priorities, and deadpan nap energy.

## Install

From the repository root:

```bash
./scripts/install-pet.sh marmalade
```

Or manually copy this folder to:

```text
${CODEX_HOME:-$HOME/.codex}/pets/marmalade/
```

## Files

```text
pet.json
spritesheet.webp
```
